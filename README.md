## One-time only set-up step:

when created new project need to remember to:
```
  $ sudo chmod 777 initialize-brackets.sh
```


## running steps
to run:
```
npm run start:pact
```
   then in a new terminal window:

```
npm run pact:seed
```
```
npm start
```
